# Exercise 5: Reading Real Code

> **Time:** 25 minutes
> **Goal:** Practice reading unfamiliar code — the #1 skill for your first week.

---

## Instructions

### Part A: Read Without Running (15 min)
Open each code file in this folder. For each file, answer:
1. What does this code do? (one sentence)
2. What language is it written in?
3. Are there any bugs or markers? (BUG, TODO, FIXME)
4. What would you change to improve it?

### Part B: Predict the Output (10 min)
For each code file, predict what would happen if you ran it.
Then actually run it and compare your prediction.

### Self-Check
If you can read and understand unfamiliar code in under 5 minutes, you're ready.
